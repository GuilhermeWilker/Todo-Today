## Roadmap

- ⏳Gerenciamento de tempo em atividades de estudo.
- 📚Melhorar produtividade do usuário.
- 🖥️Design limpo, sem poluição visual.

## Funcionalidades

- 🕢Horário em tempo real
- 🖥️Modo tela cheia
- ⏳Timer Pomodoro
- 📃Listagem de tarefas
